## Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Configure API URL** (optional)
   Create a `.env` file:
   ```
   VITE_API_URL=http://localhost:4000
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`

4. **Build for Production**
   ```bash
   npm run build
   ```
